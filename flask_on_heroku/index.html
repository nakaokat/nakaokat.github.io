Flask を Herokuで動かす

Flask: Pythonの軽量フレームワーク Heroku: Paas

Introduction 事前準備

Heokuのアカウントをつくる

python, setup-tools, pip, virtualenv, postgresql をインストールしておく

heroku login

ログインする

pip install Flask pip install gunicorn

requirements.txtを用意

pip freeze > requirements.txt

pip: pythonのパッケージを管理する

Procfileを用意

Procfile: herokuの設定用のテキストファイル

web: gunicorn {main-file-name}:app --log-file -

git 用意

git init

git add {file-name.py}

git commit -m “{commit message}"

heroku create

 git push heroku master
