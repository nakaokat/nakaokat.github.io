CSSでvisibilityプロパティを使う

2015-10-21 17:31:09 +0900 JST

CSSで要素を表示させたくない場合、display:noneだとDOMから要素自体を削除してしまう。

visibilityを使用すれば、要素自体は削除せずに要素を非表示にすることができる。

.invisible{
    visibility: hidden;
    opacity: 0;
}

transitionプロパティを使用してアニメーションで表示することも、visibilityプロパティを使うことで可能になる。

.target{
    transition: all .4s;
}
