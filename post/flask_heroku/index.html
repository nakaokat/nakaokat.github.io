1. Flask を Herokuで動かす

  • Flask: Pythonの軽量フレームワーク
  • Heroku: Paas

2. Introduction 事前準備

Heokuのアカウントをつくる

virtualenvで仮想環境をつくる（省略）

pip install {package-name} pipで必要なパッケージをインストールする

  • Flask
  • gunicorn: httpサーバー。必須
  • postgresql: データベース

pip: pythonのパッケージを管理する

heroku login

ログインする

3. requirements.txtを用意

pip freeze > requirements.txt

4. Procfileを用意

Procfile: herokuの設定用のテキストファイル

web: gunicorn {main-file-name}:app --log-file -

5. git 用意

git init

git add {file-name.py}

git commit -m “{commit message}"

heroku create

 git push heroku master

scale

heroku sacale web=1

heroku ps

確認。

heroku open

ブラウザで開く。
