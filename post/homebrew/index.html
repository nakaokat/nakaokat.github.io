Homebrewを使う

2015-09-30 01:00:31 +0900 JST

Homebrewとは

Mac OSX用のパーケージ管理システムである。

http://brew.sh/

よく使うコマンド

brew doctor：brew が正常に使えるかをチェックする。

brew install {package-name}：パッケージをインストールする。

brew outdated：新しいバージョンのあるパッケージをリストアップする。

brew update：HomebrewのFormula List をアップデートする。

brew upgrade {package-name}：{package-name}を新しいバージョンにアップグレードする。

brew upgrade：更新のあるパッケージをすべてアップグレードする。
